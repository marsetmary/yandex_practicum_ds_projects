# Проект: Прогнозирование температуры звезды

**Цель:** построить нейросеть, предсказывающую температуру звезды по её физическим характеристикам, в рамках проекта обсерватории *«Небо на ладони»*.

**Данные:** параметры звёзд — светимость, радиус, цвет, спектральный класс и др.

---

## Основные шаги
1. Загрузка и проверка данных — пропусков и дубликатов не обнаружено.  
2. EDA — анализ распределений, исправление категорий, выявление коллинеарности.  
3. Подготовка — логарифмирование, масштабирование, кодирование, преобразование в тензоры.  
4. Построение нейросетей на PyTorch (1–3 скрытых слоя).  
5. Оптимизация гиперпараметров (dropout, batch_size, learning_rate, weight_decay).  
6. Сравнение моделей и анализ ошибок.

---

## Результаты
- Лучшая модель: **3 скрытых слоя**, RMSE(test) ≈ **3416 K**.  
- Dropout=0.2, batch_size=8, lr=0.05, weight_decay=0.0001.  
- Ошибки выше при экстремальных температурах из-за ограниченности данных.

---

## Вывод
Нейросеть успешно предсказывает температуру звёзд и может применяться как базовая модель для дальнейших астрофизических исследований.
