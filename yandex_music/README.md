# Яндекс.Музыка — Москва vs Санкт-Петербург

**О чем:** Исследовательский анализ данных пользователей Яндекс.Музыки из Москвы и Санкт-Петербурга.  
Цель — сравнить активность и музыкальные предпочтения жителей двух городов.

**Данные:** `yandex_music_project.csv`

**Цели исследования**
- Проверить, зависит ли активность пользователей от дня недели и отличается ли по городам.  
- Сравнить жанры, которые слушают утром в понедельник и вечером в пятницу.  
- Оценить различия музыкальных вкусов между Москвой и Петербургом.

**Основные результаты**
- Активность по дням недели различается между городами — подтверждено.  
- В понедельник утром жанры различаются:  
  Москва — *world*, Петербург — *джаз, классика* — частично подтверждено.  
- Существенных различий в жанрах нет: *поп* и *рэп* популярны одинаково — не подтверждено.

**Инструменты:** pandas, numpy, matplotlib, seaborn
