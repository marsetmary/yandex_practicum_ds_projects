# Прогнозирование заказов такси

**Цель проекта:** построить модель, которая прогнозирует количество заказов такси на следующий час в аэропортах компании «Чётенькое такси».

---

## Задачи проекта
- Загрузить данные и выполнить ресемплирование с 10-минутных интервалов до часовых  
- Провести исследовательский анализ данных (EDA)  
- Создать новые признаки для улучшения модели (лаги, день недели, час)  
- Обучить и сравнить несколько моделей  
- Проверить модели на тестовой выборке и сделать выводы

---

## Основные результаты
- Данные агрегированы до часовых интервалов: 4416 наблюдений без пропусков  
- Выявлена ярко выраженная сезонность:  
  - Суточная с пиками в вечернее и ночное время  
  - Недельная с максимумами по понедельникам и четвергам  
- Созданы новые признаки: лаги, день недели, час  
- Построены и протестированы модели: LinearRegression, Ridge, Lasso, ElasticNet, RandomForest, XGBoost, LGBM  
- **Лучшая модель:** LGBMRegressor(iterations=100, max_leaves=31, learning_rate=0.1)  
- **Метрика на тесте:** RMSE = 35.38

---

## Выводы
- Модель LGBM эффективно учитывает линейные зависимости и краткосрочные/долгосрочные тренды через лаговые признаки  
- Склонна занижать предсказания на экстремальных значениях (низкие и высокие пики спроса)  
- Для краткосрочного прогнозирования заказов в аэропортах модель показала себя надёжной и интерпретируемой с минимальной ошибкой в пределах допустимого уровня
