# Проект: Прогноз вероятности покупки за 90 дней

**Цель:** предсказать вероятность покупки пользователя в течение 90 дней, чтобы выделять клиентов для целевых рассылок и ретаргетинга.

**Данные:** история покупок и взаимодействий пользователей (поведения и транзакции).

---

## Основные шаги
1. Изучение данных и предобработка (типизация, пропуски, кодирование).  
2. Генерация признаков поведения и агрегатов покупок (recency, frequency, monetary, last_purchase_days, total_spent и др.).  
3. Построение и подбор моделей: RandomForest, LogisticRegression, KNN, XGBoost (GridSearchCV, кросс-валидация).  
4. Работа с несбалансированными классами (undersampling / imblearn pipeline).  
5. Оценка метрики ROC-AUC и оптимизация на Recall для бизнес-целей.  
6. Тестирование и интерпретация результатов.

---

## Результат
- Лучший подход: **RandomForestClassifier**.  
- Итоговая метрика на тесте: **ROC-AUC ≈ 0.72**.  
- Модель демонстрирует практическую полезность: ~31% корректно определённых готовых к покупке клиентов при выбранной настройке (важен Recall).
