# Оценка возраста клиентов по фотографии для супермаркета «Хлеб-Соль»

**Цель проекта:** построить модель компьютерного зрения, которая по фотографии определяет приблизительный возраст клиента. Это позволяет анализировать покупки по возрастным группам, рекомендовать товары и контролировать продажу алкоголя.

---

## Задачи проекта
- Обзор и подготовка данных  
- Разработка функций для обучения модели  
- Построение и обучение модели на базе ResNet50  
- Проверка модели на тестовой выборке и оценка точности

---

## Основные результаты
- Данные: 7591 изображение с указанием возраста (`real_age`)  
- Проведен EDA: основная часть покупателей 18–40 лет  
- Применена аугментация: горизонтальное отражение, смещение, масштабирование, изменение яркости  
- Модель: ResNet50 с предобученными весами ImageNet, GlobalAveragePooling2D, Dropout 30%, полносвязный слой для регрессии  
- Обучение: 7 эпох, оптимизатор Adam (learning rate 0.0001), функция потерь MSE, метрика MAE  
- **Результат на тесте:** MAE = 6.0429

---

## Выводы
- Модель позволяет достаточно точно прогнозировать возраст клиентов по фотографии  
- Проект готов к интеграции в систему супермаркета для анализа аудитории и поддержки контроля продажи алкоголя  
- Дальнейшие улучшения могут включать увеличение объёма данных и более сложные стратегии аугментации
