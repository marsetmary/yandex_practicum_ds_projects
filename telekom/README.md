# Прогноз оттока клиентов для оператора «ТелеДом»

**Цель проекта:** обучить модель для прогноза оттока клиентов, чтобы предлагать промокоды и специальные условия для удержания абонентов.

---

## Задачи проекта
- Загрузка и анализ данных  
- Предобработка данных и создание признаков  
- Обучение моделей и подбор гиперпараметров  
- Тестирование моделей и оценка их качества  
- Формирование выводов и рекомендаций
- 
---

## Основные результаты
- Данные: таблицы contract, personal, internet, phone  
- Предобработка: названия к snake_case, даты к datetime, числовые признаки к float, пропуски заполнены, созданы новые признаки (duration, target, services, family)  
- EDA:  
  - Не ушедшие клиенты: один телефон, минимальные услуги, ежемесячная оплата  
  - Ушедшие: несколько линий, дополнительные услуги, оплата 1–2 раза в год  
- Обученные модели: RandomForest, XGBoost, LightGBM, нейросеть (2 слоя, ReLU, Dropout, BCELoss)  
- Категориальные признаки закодированы OneHotEncoder, числовые — MinMaxScaler  
- **Лучшая модель:** XGBoost (learning_rate=0.1, max_depth=5, n_estimators=500), ROC-AUC = 0.9027  
- Тестирование: ROC-AUC = 0.9259, Accuracy = 0.8839, FP = 8.4%, FN = 3.3%  
- Важные признаки: multiple_lines, services, type_Two year, duration, monthly_charges  
- SHAP подтвердил значимость признаков, duration показал сильное влияние: короткий и длинный стаж — меньше риск, средний — выше
