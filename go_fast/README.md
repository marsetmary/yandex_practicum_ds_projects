# Прокат самокатов Go Fast

**Цель:** анализ данных пользователей Go Fast и выявление возможностей для роста дохода через подписки.  
**Данные:** информация о пользователях, поездках и типах подписки за 2021 год.
---

## Основные выводы
- Пользователи без подписки: 54.4%, с подпиской: 45.6%.  
- Без подписки совершают более длинные поездки, с подпиской — более длительные по времени.  
- Средняя поездка: 18–19 минут, 2500–4000 метров.  
- Аномалии: поездки <1 м и 1 минута у пользователей без подписки.  
- Пользователи с подпиской тратят больше времени на поездки, но проезжают ≤3130 м за одну поездку.  
- Средняя помесячная выручка выше у подписчиков.  
- Для достижения KPI по подпискам нужно разослать 1169 промокодов; push-уведомление откроют ~15% пользователей.

---

## Вывод
Фокус на конверсии пользователей без подписки в подписчиков может значительно увеличить доход компании.
