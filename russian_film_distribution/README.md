# Российский кинопрокат

**Цель:** изучить тренды российского кинопроката с акцентом на фильмы с гос. поддержкой.  
**Данные:** прокатные удостоверения, сборы, поддержка, КиноПоиск (2010–2019).  

**Основные выводы:**  
- Драма и комедия — самые частые жанры среди поддержанных фильмов.  
- Средняя оценка зрителей — 6.2; лидеры по рейтингам и кассовым сборам есть, но в целом интерес умеренный.  
- Самый прибыльный год — 2017, самый убыточный — 2019.  
- Гос. поддержка с 2017 года растет, окупаемость фильмов умеренная.  

**Инструменты:** Python, pandas, numpy, matplotlib, seaborn
**Тип проекта:** EDA  
**Курс:** Яндекс.Практикум — Data Science
