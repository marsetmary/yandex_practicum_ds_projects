# Выбор локации для скважины

**Заказчик:** добывающая компания «ГлавРосГосНефть»  
**Цель:** выбрать регион для бурения новой скважины с максимальной ожидаемой прибылью и минимальными рисками.  

---

## План исследования
1. **Загрузка и подготовка данных** — проверка на пропуски и дубликаты, исследовательский анализ (EDA), изучение распределений и зависимостей признаков.  
2. **Обучение и проверка модели** — линейная регрессия с масштабированием, оценка метрики RMSE.  
3. **Подготовка к расчёту прибыли** — определение минимального объёма добычи для безубыточности, создание функции расчёта чистой прибыли.  
4. **Расчёт прибыли и рисков** — выбор 200 самых продуктивных точек, расчет прибыли и вероятности убытков методом Bootstrap.  
5. **Вывод** — определение оптимального региона для бурения.  

---

## Основные результаты
- Лучшие показатели RMSE показал **регион 2**, несмотря на относительно невысокие средние значения запасов.  
- Метод Bootstrap показал:  
  - минимальный риск убытков — 1.40%  
  - максимальная средняя ожидаемая прибыль — 463.19 млн руб.  
- **Вывод:** оптимальный регион для бурения — **регион 2**.  

---

## Вывод
Применение моделей машинного обучения и анализа рисков позволило выбрать регион с максимальной ожидаемой прибылью и минимальными финансовыми рисками для компании.
